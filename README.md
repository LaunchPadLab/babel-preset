# @launchpadlab/babel-preset

Shareable babel presets to use across projects.

## Usage:

```js
// .babelrc.js:

// To use a specific config
module.exports = {
  presets: ['@launchpadlab/babel-preset/react'],
}

// To use the default config (react)
module.exports = {
  presets: ['@launchpadlab'],
}
```
